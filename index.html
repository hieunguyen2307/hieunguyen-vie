Kỹ Năng Chuyên Môn

ROS/ROS2

Thành thạo cả ROS 1 và ROS 2. Sử dụng thành thạo các công cụ ROS như RViz, Gazebo, rqt và rosbag.

Lập Trình

Kỹ năng chuyên sâu về Python và C++. Phát triển server Python mạnh mẽ cho tích hợp robot.

Hệ Thống Đa Robot

Kinh nghiệm với hệ thống đa robot, SLAM và phát triển các gói ROS tùy chỉnh. Điều phối nhiều robot sử dụng RMF.

Điều Khiển Robot

Am hiểu về động học, động lực học robot và các thuật toán điều khiển. Ứng dụng Groot trong Nav2 cho behavior tree.

DevOps

Thành thạo Docker, Docker Swarm và Docker Compose cho triển khai ROS. Sử dụng Git và CI/CD pipeline hiệu quả.

Tiêu Chuẩn An Toàn

Hiểu biết về các tiêu chuẩn an toàn và công nghiệp cho robot, như tiêu chuẩn ISO cho hệ thống tự động.
